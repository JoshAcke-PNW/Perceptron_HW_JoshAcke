Perceptron L/T Classifier

This project implements a perceptron to classify 4x4 pixel images of L and T shapes.

Contents:

index.html
Web-based perceptron classifier deployed with GitHub Pages.

train_perceptron.ipynb
Notebook used to train perceptron and generate weights.

Annotated_data.csv
Annotated dataset containing 100 samples.

annotator.html
Used to generate L and T samples with an interactable web interface.

How it works:

1. Generate dataset
2. Train perceptron
3. Extract weights
4. Insert weights into index.html
5. Deploy with GitHub Pages
